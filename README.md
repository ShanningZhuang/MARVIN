## What is MARVIN?

MARVIN is an AI-powered assistant that truly understands your workspace. Unlike traditional AI tools that start fresh every conversation, MARVIN builds a persistent memory of your projects, preferences, and context — becoming more helpful over time.

## Features

- **Persistent Memory** — MARVIN remembers your conversations, decisions, and context across sessions
- **Smart Task Management** — Intelligent task tracking that understands priorities and dependencies
- **Workspace Awareness** — Deep integration with your development environment
- **Privacy-First** — Your data stays on your machine with optional cloud sync
